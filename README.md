# MOM System - Enterprise Management Platform 🚀

A comprehensive ERP solution designed to centralize and streamline business operations. This platform integrates multiple digital management tools into a single, intuitive interface.

> **Note:** The interface and core system are developed in Portuguese to serve local enterprise requirements.

## 🛠 Tech Stack
- **Frontend:** React with TypeScript
- **Styling:** Tailwind CSS (or CSS3)
- **Backend:** PHP (REST API)
- **Database:** MySQL

## 🌟 Key Features

### 📨 Communication & Calendar
- **Integrated Email:** Send/receive emails and link them directly to clients or suppliers.
- **Dynamic Calendar:** Personal and shared views with event reminders based on permission levels.

### 💰 Financial Management (CC Clientes & Fornecedores)
- **SAFT Integration:** Import SAFT files for automated accounting.
- **Current Accounts:** Full control over payments, discounts, and bank transfer exports (XLSX).
- **Billing Control:** Track paid vs. overdue invoices in real-time.

### 🏗 Operations & Production
- **Service Orders (OS):** End-to-end production tracking with document management.
- **Proposals & Quotes:** Integrated budgeting system with status tracking (Draft, Sent, Awarded).
- **Task Management:** Real-time planning and task assignment with reminder systems.

### 👥 HR & User Permissions
- **Granular RBAC:** Different access levels (Admin, Director, Coordinator, Collaborator).
- **Labor Cost Automation:** Hourly rate assignment per employee to automate project profitability analysis.

## 📈 Impact
The MOM system bridges the gap between raw data and business intelligence, replacing scattered spreadsheets with a unified production-ready infrastructure.
